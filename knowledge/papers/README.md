# Paper Cards

This directory stores literature-derived structured knowledge for the agent.

## Files
- `index.json`: flat index of all ingested papers.
- `cards/*.json`: one card per paper.

## Notes
- Cards are auto-generated from the first two PDF pages and should be treated as first-pass summaries.
- They are intended for planning support, feature-set design, and reasoning support, not as definitive paper reviews.
- The most relevant cards for the current TS-SatFire workflow should be manually reviewed and enriched over time.
